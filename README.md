# 🏎️ GTA Online Tracker - Pre-release v0.9.0

¡Bienvenido a **GTA Online Tracker**! Una aplicación multiplataforma diseñada para el jugador dedicado de GTA Online que quiere llevar un control total de su imperio criminal. 

Desarrollada con **Flutter**, esta app combina funcionalidad de alto nivel con una interfaz limpia.

## ✨ Características Principales
* **🏛️ Catálogo de Propiedades:** Gestión de 232 propiedades únicas, desde apartamentos de lujo en Eclipse Towers hasta búnkeres en Paleto Bay.
* **🚗 Gestión de Garajes:** Organiza tu colección de vehículos por ubicación real.
* **🌐 Soporte Multi-idioma (En desarrollo):** Traducida completamente al Español e Inglés.
* **📊 Base de Datos Local:** Uso de SQLite para que tus datos estén siempre seguros en tu dispositivo.
* **🤖 IA Ready:** Estructura preparada para importar datos mediante capturas de pantalla o texto plano.
